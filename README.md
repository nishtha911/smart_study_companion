# TrackSpace: The Smart Study Companion

## Overview

### TrackSpace is a custom, feature-rich dashboard designed to move academic study planning beyond simple to-do lists. This application translates my semester 5 subjects into an actionable, prioritized plan using MySQL data management and real-time urgency calculations. The goal is to enforce efficient, proactive studying based on deadline, unit, and perceived difficulty.

## Tech Stack

### This project utilizes a modern, robust, and scalable Python web stack:

- Backend: Python 3 + Flask (Micro-framework)

- Database: MySQL (Relational Database Management System)

- Frontend: HTML5 and Custom CSS

- Design: Custom Dark Mode with Emerald Green "Liquid Glass" Aesthetic
